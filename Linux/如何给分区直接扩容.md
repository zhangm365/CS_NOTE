# Ubuntu 分区扩容

## 1. 查看分区

```bash
# 查看扩容前的分区大小
$ df -hl
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda3       125G  109G   11G  92% /

```

## 3. 刷新分区

```bash
$ sudo resize2fs /dev/sda3

# 查看分区是否扩容成功
$ df -hl
/dev/sda3       251G  113G  128G  47% /

```
