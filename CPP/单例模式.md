# C++单例模式

## 原理：私有化该类的构造和析构函数，通过一个公有的静态方法来获取该类的唯一实例。

```c++
class Singleton
{
    private:
    	Singleton();
    	~Singleton();
  
    public:
    	static Singleton* getInstance()
        {
            static Singleton instance;	// 局部静态变量，创建一个单例对象
           
            return &instance;
        }
}
```

因为类的构造函数是私有的，故不允许在类外创建该类的实例。此时，只能通过公有的静态方法访问该类的唯一实例。
