# map & multimap

--------------------

## 1. 红黑树

红黑树是一颗高度平衡二叉树，除了满足 **二叉搜索树** 的约束外，还需满足以下要求：

1.  每个节点要么是红色，要么是黑色；
2.  所有 `NIL` 叶子都被认为是黑色的；
3.  如果一个节点是红色的，那么它的两个孩子节点必是黑色的；
4.  从给定节点到其任何后代 `NIL` 叶都经过相同数量的黑色节点；

## 2. map

`map` 可以是用运算符 `[]` 进行插入数据操作

## 3. multimap

`multimap` 不可以使用运算符 `[]` 进行插入操作，只能通过函数 `insert` 插入数据。
