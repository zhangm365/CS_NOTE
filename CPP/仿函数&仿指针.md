# 一. 仿指针

## 1. 使用模板类去实现指针的作用，使得指针拥有更多的功能。C++中的智能指针就是这样的实现原理。

## 2. 实现代码pointer-like class

```c++
template <class T>
class shared_ptr
{
    public:
    	T& operator*() const
        {
            return *px;
        }
    	T* operator->() const
        {
            return px;
        }
    	shared_ptr(T* p) : px(p){}
    	~shared_ptr(){delete px;}	//释放指针, 防止内存泄露
    
    private:
    	T* px;
};
```
