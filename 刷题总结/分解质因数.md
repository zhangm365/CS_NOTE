
# 分解质因数的代码模板

1. 分解质因数

将给定的变量分解为质因数，并按质因数从小到大的顺序输出每个质因数的底数和指数


```c++

/* 
 * @param x: 待求解的值
 * 
 * 
*/
void divide( int x )
{
    for( int i = 2; i <= x/i; ++i )
    {

        if( x % i == 0 )
        {   
            int exp = 0;    // 指数
            while( x % i == 0 )
            {
                x /= i;
                exp++;
            }
            cout << i << " " << exp << endl;    // 输出质因数的底数和指数
        }

    }

    if( x > 1 ) cout << x << " " << 1 << endl;
    cout << endl;

}
```
